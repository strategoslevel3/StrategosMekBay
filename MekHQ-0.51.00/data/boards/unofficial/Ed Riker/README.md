## To use:

Unzip and Place in `<megamek root>/data/boards` Folder.

To create a 2x2 island

```
0: IslandNW
1: IslandNE
2: IslandSW
3: IslandSE
```

To create a 3x3 island

```
0:IslandNW
1:IslandN01Cliffshore
2:IslandNE
3:IslandW01Harbor
4:[Suprise] (Or what ever map you want)
5:IslandE01
6:IslandSW
7:IslandS02BasePlain
8:IslandSE
```
